FilFinder_Results

Hosts larger images created from the FilFinder output. Currently, this only includes results from the Herschel Gould Belt Survey (preliminary, publicly released data at 350 micro-m).

If you have used FilFinder, please consider adding similar figures to the gallery! To add your own figures, create a fork of the repository. Clone your fork to a local location, add your figures, commit the changes, then push to your fork. Finally, submit a pull request to have it included with the rest of the collection!

We hope for this site to contain examples of running FilFinder on a range of datasets, making it easier for members of the community to decide if FilFinder is an appropriate analysis tool for their data.

The documentation for FilFinder is here, and the repo is here.
